# Changelog

All notable changes to TEMPORA will be documented in this file.

## 0.1.0-alpha - Unreleased

### Added

- Initialized Python package with Docker, CI, pytest, Ruff, and Mypy.
- Added synthetic circle, torus, Lorenz, and Rossler data generators.
- Added finite perturbations for time warping, Gaussian noise, and missing
  segments.
- Added contractive CTRNN core with spectral projection and Jacobian diagnostics.
- Added projected Oja-style plasticity and a small circle training smoke loop.
- Added persistent-homology metrics and finite point-cloud topology comparison.
- Added stability and invariance diagnostics with JSON and figure outputs.
- Added GRU, unconstrained Neural ODE, and reservoir baselines.
- Added CI-safe synthetic smoke benchmark with `metrics.json`, figures, and a
  Markdown report.
- Added benchmark persistence-diagram figures for input and latent point clouds.
- Added reproducible benchmark artifact snapshots for resolved configs and
  dataset-level model checkpoints.
- Added benchmark report sections for run metadata, artifact paths, and
  dependency versions.
- Added a CI smoke-benchmark gate with read-only workflow permissions.
- Added automated local Markdown link checks for README and docs.
- Added example YAML configs for synthetic datasets and the contractive CTRNN.
- Added machine-readable sufficient-contraction proof certificates.
- Added machine-readable projected-learning stability certificates.
- Added machine-readable empirical topology comparison certificates.
- Added benchmark contraction certificates in metrics and reports.
- Added benchmark projected-learning stability certificates for the last
  plasticity update.
- Added benchmark topology comparison certificates with a configured H1
  persistence-distance threshold.
- Added benchmark certificate-summary counts and failure listings.
- Added benchmark certificate-gate policy for required proof-adjacent checks.
- Added a CLI check for benchmark certificate gates in CI.
- Documented the benchmark certificate gate in README and release checks.
- Added reusable checkpoint save/load helpers for contractive CTRNN artifacts.
- Added conservative theory documentation for contraction, projected learning
  stability, and persistence-diagram comparison.

### Verification

- Current release-prep branch requires:
  - `docker compose run --rm tempora`
  - `docker compose run --rm tempora python scripts/train_synth.py --config configs/benchmark_smoke.yaml`

### Notes

- Benchmark outputs are generated under `outputs/` and are not committed.
- No benchmark result should be reported unless it was generated by an actual
  run and reviewed with its config and seed.
- License is pending project-owner selection.
